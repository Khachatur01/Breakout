# Breakout
Classic Breakout game for Unix like system's console
___
Controls
  
    a - left
    
    d - right
  
    p - pause / resume
___
Run "g++ main.cpp -lncurses" to compile.
___
cmake .

make

./breakout

to compile with cmake and run.
